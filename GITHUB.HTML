Project

Weaver Management System

The Weaver Management System is a Spring Boot–based application designed to streamline weaver operations. It allows tracking of employee working hours, production output, and stock details in an organized way. The system automatically calculates payments based on employee output, reducing errors and saving time. Data is securely managed using MySQL, ensuring reliability and easy retrieval. A user-friendly interface built with HTML, CSS, and Java makes the system simple to use. Overall, the project improves efficiency by digitizing traditional manual processes in weaver management.

Tech Used: Java, Spring Boot, MySQL, HTML, CSS, JDBC/Spring Data JPA, Maven, Tomcat Server
